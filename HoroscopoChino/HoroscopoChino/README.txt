Horóscopo Chino 
===============
===============


Lenguaje e IDE 
==============
Java 1.8 - Netbeans 8.2

Contexto
========
El ser humano es curioso por naturaleza, asi que este pequeño programa 
permite cubrir esa curiosidad de aquel grupo de personas que quieren saber 
a que signo del horóscopo Chino pertenecen.

Elementos que tratan de ser cubiertos con éste ejemplo
======================================================
-Software útil para un determinado grupo de personas, cubre la necesidad de conocer a que signo se pertenece en el 
horoscopo Chino dado el año de nacimiento.
-Software que continue siendo util en el tiempo, esta organizado de manera que el modelo es independiente
de la interfaz de usuario (en este caso solo consola en el metodo MAIN) con el tiempo puede mejorarse la IU y la información que 
se muestra al usuario el modelo es reutilizable.
- Facilidad de mantenimiento, utiliza convenciones del lenguaje documentacion del codigo, notacion camelCase, nombres
de variables descriptivas.
